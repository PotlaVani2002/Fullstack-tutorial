HTML Layout and Comments

HTML documents typically have a standard structure, including <!DOCTYPE >, <html>, <head>, and <body> elements.

The <!DOCTYPE> declaration specifies the document type and version.

The <html> element is the root element of the document.The HTML document itself begins with <html> and ends with </html>.

The <head> section contains metadata, while the <body> section contains visible content.

Document Type Declaration

Declares the document type as HTML5.
It must be the very first line in the document.
The <!DOCTYPE> declaration is not case sensitive.
Helps the browser correctly interpret and render the HTML code. 

<!DOCTYPE html>

HTML Tag

The root element of the document.
The lang attribute specifies the language, improving accessibility and SEO. 

<html lang="en">

Head Section

Contains metadata about the document (e.g., character encoding, title, and viewport settings).
Metadata is not displayed on the web page but is essential for proper rendering and functionality. 

<head> ... </head>

Body Section

The <body> element contains all the visible content rendered on the page, such as text, images, and links. 

<body> ... </body>

Comments

HTML comments are not displayed in the browser, but they can help document your HTML source code.

< !-- Comments -->

Example Code

<!DOCTYPE html>  <!-- Declaration of the document type as HTML5 -->
<html lang="en"> <!-- Root element with language as English -->
<head> <!-- Metadata section -->
    <meta charset="UTF-8"> <!-- Specifies that the character encoding for the document is UTF-8, which includes a wide range of characters from various languages. -->
    <meta name="viewport" content="width=device-width, initial-scale=1.0"> <!-- The 'viewport' meta tag is used to control how the document is displayed on different devices. 'width=device-width' makes the document width match the screen width, and 'initial-scale=1.0' sets the initial zoom level to 100%.-- >
    <title>Document</title> <!-- Sets the title of the document, which appears on the browser tab or window.  -->
</head>
<body> <!-- Visible page content -->
    <p>Hello World</p> <!-- A paragraph element -->
</body>
</html>
