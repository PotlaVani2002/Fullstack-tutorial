HTML Headings

HTML headings are used to define the structure and hierarchy of a document.

HTML provides six levels of headings, <h1> (Highest) through <h6> (Lowest), used to structure content hierarchically.

Use headings to provide a clear and logical structure to your content.

Search engines use headings to understand the hierarchy of your content and improve SEO.

Heading Level 1 (h1)

Used for the most important titles or page headers. Default font size: ~32px.

Heading Level 2 (h2)

Typically used for subsections under the main header. Default font size: ~24px.

Heading Level 3 (h3)

For smaller subsections. Default font size: ~18.72px.

Heading Level 4 (h4)

For content of lesser importance. Default font size: ~16px.

Heading Level 5 (h5)

Used for minor content headings. Default font size: ~13.28px.

Heading Level 6 (h6)

Smallest and least important heading. Default font size: ~10.72px.

Example Code


    <h1>Heading level 1</h1>
    <h2>Heading level 2</h2>
    <h3>Heading level 3</h3>
    <h4>Heading level 4</h4>
    <h5>Heading level 5</h5>
    <h6>Heading level 6</h6>

Best Practices

  • Use <h1> only once per page, typically for the main title.
  • Maintain a logical order (e.g., avoid skipping from <h1> to <h4>).
  • Use headings to improve accessibility and SEO.
