HTML Anchor Tag

The <a> tag creates hyperlinks that connect web pages, files, emails, or phone numbers. Below are examples demonstrating its possibilities:

Basic Link

<a href="https://www.example.com">Visit Example.com</a>
Visit Example.com

The href attribute specifies the URL the link points to.

Open Link in a New Tab

<a href="https://www.example.com" target="_blank">Visit Example.com in a new tab</a>
Visit Example.com in a new tab

The target="_blank" attribute opens the link in a new tab or window.

Email Link

<a href="mailto:contact@example.com">Send an Email</a>
Send an Email

Use mailto: followed by an email address to create an email link.

Telephone Link

<a href="tel:+1234567890">Call Us</a>
Call Us

Use tel: followed by a phone number to create a telephone link.

Link to a Section on the Same Page

<a href="#section1">Go to Section 1</a>
Go to Starting of the page

Use #id to link to an element with a corresponding id attribute.

This is Section 1. You have successfully navigated here!

Link to a File

<a href="./files/example.pdf" download>Download Example PDF</a>
Download Example PDF

The download attribute suggests downloading the file instead of opening it.

Link with Tooltip

<a href="https://www.example.com" title="Go to Example.com">Hover over me</a>
Hover over me

The title attribute displays a tooltip when the user hovers over the link.

Styled Link

<a href="https://www.example.com" style="color: red; font-weight: bold;">Custom Styled Link</a>
Custom Styled Link

Use the style attribute or CSS for custom link styling.

Link with Rel Attribute

<a href="https://www.example.com" target="_blank" rel="noopener noreferrer">Secure External Link</a>
Secure External Link

The rel="noopener noreferrer" attribute improves security and prevents the new page from accessing the window.opener object.

Images can be linked, just like text.

<a href="./images/shinchan.png" > 
        <img src="./images/shinchan.png"
        alt="Click me for a larger view" />
</a>
