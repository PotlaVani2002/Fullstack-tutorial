HTML Formatting Tags

1. Bold Text

<b>: Makes text bold. Visually bold without semantic meaning.

Example: < b >This is bold text using <b>.

<strong>: Makes text bold and adds semantic importance.

Example: < strong >This is important text using <strong>.

2. Italic Text

<i>: Makes text italic. Visually italic without semantic meaning.

Example: <i>This is italic text using <i>.

<em>: Emphasizes text with semantic meaning.

Example: <em>This is emphasized text using <em>.

3. Underline

<u>: Underlines text.

Example: <u>This is underlined text using <u>.

4. Strikethrough

<strike>: Makes text strikethrough (deprecated in HTML5).

Example: < strike >This is strikethrough text using <strike>.

                   OR

< s >This is strikethrough text using <s>.

<del>: Represents deleted text semantically.

Example: <del>This is deleted text using <del>.

5. Inserted Text

<ins>: Represents inserted text semantically and is visually underlined.

Example: <ins>This is inserted text using <ins>.

6. Superscript and Subscript

<sup>: Makes text superscript (useful in math).

Example: E= mc <sup> 2 </sup>

Output: E = mc2

<sub>: Makes text subscript (useful in chemistry).

Example: H <sub> 2 </sub> O

Output: H2O
