HTML Image Tag

Description

The <img> tag is used to insert images into an HTML document. It is a **void element**, meaning it does not have a closing tag.

Attributes of the <img> Tag

  • src: Specifies the path or URL of the image file. This is a required attribute.
  • alt: Provides alternative text for the image, which is displayed if the image cannot load. This is important for accessibility.
  • width and height: Define the width and height of the image in pixels or percentage.
  • title: Adds a tooltip when the user hovers over the image.
  • loading: Specifies lazy loading behavior for performance optimization. Values can be lazy, eager, or auto.

Example Code

 <img src="example.jpg" alt="A beautiful example image" width="400" height="300" title="Hover to see this text">

Notes

  • The <img> tag does not have a closing tag. Example: <img>.
  • Always include the alt attribute for accessibility and better SEO.
  • Use properly compressed images to improve page load time.
  • For responsive design, use CSS or the max-width property to adjust image sizes.
